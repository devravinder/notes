# Eclipse Shortcuts

| Shortcut | Description |
|---|---|
| `alt+shf+r` | Rename |
| `sout` or `syso` | `System.out.println();` |
| `ctrl+space` | Code Assist |
| `ctrl+A`, `ctrl+i` | Indent |
| `tab` | Indent |
| `shift+tab` | Outdent |
| `ctrl+f` | Find |
| `f3` | Jump to method definition |
| `alt+shift+s` | Code generator |
| `ctrl+shf+p` | Go to corresponding bracket |
| `alt+shift+z` | Wrap in block |
| `ctrl+shif+/` | Comment |
| `ctrl+shif+\` | Uncomment |
| `ctrl+sfif+x` | Upper case |
| `ctrl+sfif+y` | Lower case |
| `ctrl+shift+l` | List all short keys |
| `ctrl+shit+f4` | Close all |
| `ctrl+f11` | Run |
| 3 times click | Select line |
| 2 times click | Select word |
| `ctrl+l` | Go to line |
